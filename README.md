# 🎮 Kontrol Permainan

Berikut adalah daftar tombol untuk mengontrol permainan:

## 🕹️ Arah Pergerakan
- **W** → Bergerak ke atas
- **A** → Bergerak ke kiri
- **S** → Bergerak ke bawah
- **D** → Bergerak ke kanan

## 🔫 Aksi
- **Enter** → Menembak
- **Space** → Mengulang permainan (Restart)

Selamat bermain! 🚀
